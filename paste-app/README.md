# Project Overview

React Paste App is a simple web application inspired by Pastebin. It allows users to create, edit, and copy text-based pastes. It uses React for the frontend, Redux for state management, and React Router DOM for navigation.

The app is styled using Tailwind CSS and provides a dark-themed interface for a modern look.

# Features

* Create new pastes with a title and content.

* Edit existing pastes.

* Copy paste content to clipboard with a single click.

* Dynamic routing to edit specific pastes.

* Responsive design for desktop and mobile devices.

* Notification system for user actions (e.g., copy success).

# Tech Stack

* React – Frontend framework for building UI.

* Redux Toolkit – Efficient state management.

* React Router DOM – Page navigation and routing.

* Tailwind CSS – Utility-first CSS framework.

* React Hot Toast – Real-time notifications.

* React Icons – Icon library for UI elements
